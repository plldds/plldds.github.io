#### 安装

```
brew tap majd/repo
brew install ipatool
```

#### 配置用户信息

```
sudo ipatool auth login -e xxxxx@qq.com -p xxxxxx
```

#### 查看用户信息

```
sudo ipatool auth info
```

#### 搜索

```
sudo ipatool search -l 5 weibo
```

#### 下载(cd到下载目录)

```
sudo ipatool download -b com.sina.weibo -o weibo.ipa
```
